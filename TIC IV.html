TIC IV

El diseño gráfico y la creación de sitios web han llegado a ser aspectos de gran importancia en el mundo publicitario, pues su función principal es causar una buena impresión, captar la atención y generar confianza en los visitantes mediante su diseño o aspecto visual. Crear una página web y sus elementos multimedia es un arte que permite proyectar un conjunto de objetos visuales visuales para organizarlos, de tal manera que logren atraer a las personas. En esta materia se divide en dos submódulos: en el primero conocerás las etiquetas básicas de HTML para crear sitios web y poner textox, títulos, imágenes, tablas colores, etc., así como las etiquetas de diseño de CSS y el uso de sowftware de aplicación; en el segundo, aprenderás las bases del diseño digital como los tipos de gráficos y el uso de software de aplicación, conocimientos que se aplican a la creación de diversos sitios web.
